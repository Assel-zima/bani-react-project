# ДА Бани — React Digital Product

Готовый индивидуальный проект по React: сайт строительной компании бань.

## Что реализовано по требованиям

- 6+ страниц: Главная, Услуги, Детальная услуга, Каталог, Детальный товар, Калькулятор, Отзывы, Контакты, NotFound.
- Динамические страницы через `useParams`: `/services/:id` и `/catalog/:id`.
- 3 запроса к серверу через `fetch`: `GET /products`, `GET /products/:id`, `GET /users`.
- React Context: переключение светлой/темной темы, избранное, модальное окно заявки.
- 3+ интерактивных элемента: переключение темы, поиск услуг, избранное, калькулятор, форма, модальное окно.
- Дополнительный инструмент: `classnames`.
- Использованы изображения из архива сайта бань.

## Запуск

```bash
npm install
npm run dev
```

## Сборка
